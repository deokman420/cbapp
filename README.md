CB App Setup for Enterprise Browser Goal: Run CB App in Enterprise browser to enable notepad-like functionality (Enterprise blocks copying text externally).Prerequisites:-Chrome (to download CB App)-Enterprise browser (to run CB App)-SSO access for Enterprise login Steps:1.In Chrome:-Go to phbeks.com/cbapp, download cbapp.zip.-Extract cbapp.html to Documents folder.2.In Enterprise:-Log in via SSO.-Open cbapp.html (right-click > Open with Enterprise if not default).3.Optional Bookmark:-Enable Bookmarks Bar (three-dot menu > Bookmarks > Show Bookmarks Bar).-Drag CB App bookmark from Bookmarks Menu to Bookmarks Bar.Troubleshooting:-App not working? Ensure SSO login and cbapp.html opened in Enterprise. Set Enterprise as default if needed.-Bookmark missing? Verify Bookmarks Bar is enabled and CB App is bookmarked.Note: Keep cbapp.html in Documents for easy access. Must open in same Enterprise session as SSO login.
